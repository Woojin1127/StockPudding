# Stock Pudding — PRD

> 어려운 주식 지표를 주린이도 떠먹기 쉽게 풀어주는 종목 분석 서비스

**버전** 0.1 (MVP) · **작성일** 2026.06.24 · **상태** 초안

---

## 1. Background (배경)

주식에 관심은 있지만, 지표가 무엇을 의미하는지 모르고 무엇을 봐야 할지 막막한 초보자가 많음.

- RSI, 이평선, PBR 같은 용어를 봐도 "그래서 좋은 거야 나쁜 거야?"를 모름
- 증권사 앱은 정보가 너무 많아서 초보자에겐 오히려 부담
- 차트 앱은 지표를 보여주기만 하고 의미를 설명하지 않음

결국 초보자는 감으로 사고, 불안한 채로 들고 있게 됨. 이 지점을 파고들어, **객관적 지표를 쉽게 풀어서 떠먹여주는** 서비스가 필요함.

---

## 2. Problem (문제 정의)

- **지표를 봐도 해석이 안 됨** — 숫자만 있고 "이게 좋은 신호인지"가 없음
- **무엇을 봐야 할지 모름** — 종목 하나를 판단할 때 어떤 지표를 봐야 하는지 막막함
- **정보 과부하** — 기존 도구는 전문가 기준이라 초보자에겐 진입장벽이 높음

기존 도구 대부분이 "데이터를 잔뜩 보여주는 스크리너" 방향임. 초보자에게 필요한 건 그 반대로, **걸러낸 핵심 지표를 쉬운 말로 해석해주는 도구**임.

---

## 3. Goal (목표)

초보자가 **종목 하나를 검색하면, 그 종목이 지금 어떤 상태인지 5초 만에 이해**할 수 있게 하는 것.

세부 목표:
- **쉬운 해석** — 지표를 모르는 사람도 "왜 중요한지"가 바로 보이게
- **객관성** — 감정/뉴스 빼고 객관적 지표 기반 (MVP 단계)
- **재미** — 좋아요/싫어요, 랭킹으로 가볍게 즐길 수 있게

---

## 4. Target (타겟 사용자)

**주린이 (투자 경력 0~2년)**

- 지표 용어는 들어봤지만 의미는 모름
- 종목을 살 때 무엇을 봐야 할지 기준이 없음
- 복잡한 전문가용 도구는 부담스러움
- 쉽고 직관적인 설명을 원함

---

## 5. User Journey (사용자 여정)

**메인 흐름 — 종목 분석**

1. 앱 진입
2. 검색바에 종목명 또는 코드 입력 (예: 삼성전자 / 005930)
3. 검색
4. 분석 결과 확인
   - 종합 점수 + 신호등 판정 ("지금은 조심할 때")
   - 지표별 카드 (값 + 쉬운 설명 + 신호)
   - 눈에 띄는 신호 요약
   - 한줄 진단
5. "이 분석 도움됐어요 / 별로예요" 투표
6. (선택) 다른 종목 검색

**서브 흐름 — 인기 분석 보기**

1. 메인 하단에서 "오늘 많이 본 종목" / "도움된 분석 TOP" 확인
2. 클릭해서 해당 종목 분석 + 투표 결과 확인
3. 직접 투표 참여

---

## 6. 필수 기능 및 우선순위 (Core Features & Priority)

### P0 — MVP (반드시)

**6.1 종목 검색 & 분석**
- 종목명(한글) 또는 코드(숫자)로 검색
- 실시간 데이터로 지표 계산 후 결과 표시
- 사용 지표: RSI, 52주 고점 대비 낙폭, 이평선 대비 현재가, 거래량(평균 대비), PBR, 부채비율
- 결과 구성: 종합 점수(0~100, 게이지) + 신호등 판정 + 지표별 카드 + 신호 요약 + 한줄 진단

**6.2 리포트 저장**
- 분석 결과를 고유 ID로 DB에 저장 (투표 추적, 공유에 사용)

**6.3 좋아요 / 싫어요 투표**
- 결과 하단에 "도움됐어요 / 별로예요" 버튼
- DB에 기록, 기기 단위 중복 투표 방지
- 클릭 시 수치 즉시 반영

**6.4 캐싱**
- 같은 종목을 짧은 시간 내 재검색하면 저장된 데이터 재사용 (API 호출 절약)
- 일정 시간 지나면 자동 갱신

### P1 — 첫 배포 후

**6.5 인기 분석 랭킹**
- "도움된 분석 TOP" (좋아요 순)
- "별로였던 분석 TOP" (싫어요 순, 가벼운 유머톤)
- 종목명, 점수, 좋아요/싫어요 수 표시

**6.6 인기 검색어**
- "오늘 많이 본 종목" (DB 집계)
- "내 최근 검색" (로컬 저장)

### P2 — 다음 버전

- 로그인 & 관심 종목 저장
- 알림 (관심 종목 위험 신호)
- 시간대별 정확도 분석 (다음날 주가와 비교 검증)

---

## 7. Out of Scope (이번 버전 제외)

- **뉴스/감정 분석** — MVP는 객관적 지표만
- **AI 생성 진단** — 해석 텍스트는 규칙 기반으로 작성
- **백테스트** — 과거 전략 검증은 나중에
- **미국/해외 주식** — 한국 주식(코스피, 코스닥)만
- **상세 재무 분석** — PBR, 부채비율 정도만 (손익계산서 깊은 분석 X)
- **포트폴리오 관리** — 보유 종목 추적 X
- **매매 신호** — "지금 사세요/파세요" 같은 강한 권유 절대 X
- **로그인/회원 기능** — P2로 미룸
